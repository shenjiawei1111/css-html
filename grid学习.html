CSS Grid布局系统学习

掌握Grid布局的核心概念和常用属性

1. 基础网格 (grid-template-columns)

grid-template-columns定义列的数量和宽度，fr单位表示剩余空间的比例分配。

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3列等宽 */
  grid-gap: 10px;
}

2. 行高设置 (grid-template-rows)

grid-template-rows定义行的高度，可以使用固定值或比例。

.container {
  display: grid;
  grid-template-columns: 1fr 1fr; /* 2列 */
  grid-template-rows: 100px 200px; /* 第一行100px，第二行200px */
  grid-gap: 10px;
}

3. 区域布局 (grid-template-areas)

grid-template-areas通过命名区域来定义布局，使代码更具可读性。

.container {
  display: grid;
  grid-template-columns: 150px 1fr 150px;
  grid-template-rows: 80px 1fr 80px;
  grid-template-areas:
    "header header header"
    "sidebar main ads"
    "footer footer footer";
}

.header { grid-area: header; }

4. 自动行高 (grid-auto-rows)

grid-auto-rows定义隐式创建的行的默认高度，适用于项目数量不确定的情况。

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3列 */
  grid-auto-rows: 80px; /* 自动创建的行高度为80px */
  grid-gap: 10px;
}

5. 自动流动 (grid-auto-flow)

grid-auto-flow控制项目的排列方式，dense值可以填充空白区域。

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: dense; /* 紧凑排列，填充空白 */
  grid-gap: 10px;
}

.wide { grid-column: span 2; } /* 跨2列 */
